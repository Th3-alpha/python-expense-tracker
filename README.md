# Python Expense Tracker

A command-line personal finance application built with **Python**, **Pandas**, and **Matplotlib** that enables users to manage expenses, maintain persistent records, and analyze spending patterns through reports and visualizations.

---

## Features

- Add new expenses (date, category, description, amount)
- View all expenses in a clean table
- Edit existing expenses by ID
- Delete expenses by ID
- View a summary report (total + category breakdown)
- Visualize spending by category with a bar chart (Matplotlib)
- Data stored persistently in `expenses.csv`

---

## Technologies Used

- **Python 3**
- **Pandas** – for data handling and table formatting
- **Matplotlib** – for chart visualization
- **CSV** – for local data storage

Run the application using:


## Usage

When the application starts, the following menu is displayed:

==== Expense Tracker ====
1. Add Expense
2. View Expenses
3. Edit Expense
4. Delete Expense
5. View Summary Report
6. Exit

Users can:

- Add new expense records
- View previously recorded expenses
- Modify existing records
- Delete records that are no longer needed
- Generate spending reports and visualizations
